1 Create Snipcart Account

Action Required

Snipcart handles the shopping cart, checkout, and payment processing for your website. You need a free account to get started.

1

Sign up for Snipcart

Create a free account. No credit card needed for test mode.

Create Snipcart Account ↗
2

Get your API Key

After signing in, go to: Account → API Keys. Copy the Public API Key (starts with something like MjM...).

Open API Keys Page ↗
3

Paste your API Key below

This is the PUBLIC key (safe to use in HTML). Never share your SECRET key.
Saved! See deployment instructions below.
After saving: The API key will be stored locally. To make it live, you need to update the code. See the generated code snippet below after saving.

2 Connect Payment Method

Action Required

Snipcart supports Stripe (credit cards), PayPal, Square, and more. You need to connect at least one payment gateway.

1

Option A: Stripe (Recommended - Credit/Debit Cards)

Stripe is the easiest way to accept credit cards. Create a Stripe account if you don't have one.

Create Stripe Account ↗ Connect in Snipcart ↗
2

Option B: PayPal

Connect PayPal as an additional or primary payment method.

Connect PayPal in Snipcart ↗
Important: In Snipcart dashboard, go to Account → Payment Gateway to connect Stripe or PayPal. Your store won't accept real payments until this is done and you switch from Test to Live mode.
3

Switch to Live Mode

When ready to accept real payments, switch from Test Mode to Live Mode in Snipcart dashboard. Make sure to update the API key (Live mode has a different key).

Snipcart Dashboard ↗

3 Connect Domain (DNS)

Action Required

Connect your custom domain (e.g., giftelix.com) to the Vercel deployment.

Current URL: giftelix-web.vercel.app
Vercel Project: Open Domain Settings ↗
1

Add domain in Vercel

Go to Vercel Dashboard → giftelix-web → Settings → Domains. Add your domain (e.g., giftelix.com).

Open Vercel Domain Settings ↗
2

Update DNS Records

After adding the domain in Vercel, you'll get DNS records to add at your domain registrar. Add these records:

Type Name Value A @ 76.76.21.21 CNAME www cname.vercel-dns.com
* These are Vercel's default DNS values. Verify the exact values shown in your Vercel domain settings.
3

Where to add DNS records

Log in to your domain registrar (where you bought the domain) and update DNS:

Namecheap ↗ GoDaddy ↗ Cloudflare ↗ Google Domains ↗
4

Update Snipcart Domain

After DNS is live, update your domain in Snipcart: Account → Domains & URLs. Add your custom domain so checkout works correctly.

Snipcart Domain Settings ↗
DNS propagation takes 5 minutes to 48 hours. Use dnschecker.org to check progress.

4 Deploy API Key Changes

After Step 1

After getting your Snipcart API key, the code needs to be updated. Here's what needs to change:

Find and replace in ALL HTML files:
Replace YOUR_SNIPCART_PUBLIC_API_KEY with your actual API key.

Your API Key:

data-api-key=""

Files that need updating:

index.html products/gfx-001.html products/gfx-002.html ... (all 52 product pages)
A

Option A: GitHub (Recommended)

Edit the files on GitHub directly. Changes auto-deploy to Vercel.

Open GitHub Repo ↗
B

Option B: Ask Developer

Share your API key with the developer. They'll update all files and push the changes.
