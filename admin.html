🚀 Quick Setup Guide
1

Create Snipcart Account

Sign up at snipcart.com → go to Account → API Keys → copy the Public API Key

2

Add API Key to Vercel

Go to Vercel → Settings → Environment Variables
Add: Name = SNIPCART_API_KEY   Value = your API key
Click Save → site auto-redeploys with the key!

3

Connect Stripe (accept payments)

Create Stripe account → go to Snipcart → Payment Gateway → connect Stripe

4

Add Domain

Go to Vercel → Domains → add your domain (e.g. giftelix.com)
Then send setup.html link to your friend for DNS records

5

Go Live!

In Snipcart: switch from Test Mode to Live Mode → update API key in Vercel with the Live key
Add your domain in Snipcart → Domains
