Step 1 of 5

Create Your Shopping Cart Account

Snipcart powers the checkout on your website. Create a free account to get started (no credit card needed).

1 Sign up for Snipcart

Click below to create your free account:

Create Snipcart Account ↗

2 After signing up, get your API Key

In Snipcart dashboard: click your name (top-right) → API Keys. Copy the Public Test API Key.

Open API Keys Page ↗
🔒 Security note: The Public API Key is safe to use — it only allows adding items to cart. Your secret key and payment info stay protected in Snipcart's secure dashboard.
Step 2 of 5

Connect API Key to Website

Paste your Snipcart Public API Key below. It will be securely saved and the website will automatically update.

Starts with letters/numbers, about 36 characters long. Example: MjM4N2...
🔐 Your key is encrypted and stored as a server-side environment variable. It is never visible in the website source code.
Step 3 of 5

Connect Payment Method

Connect Stripe or PayPal so you can accept real payments from customers.

1 Create a Stripe account (if you don't have one)

Stripe handles credit/debit card payments. Free to sign up, you only pay per transaction.

Create Stripe Account ↗

2 Connect Stripe to Snipcart

In Snipcart: go to Account → Payment Gateway → click Connect next to Stripe.

Open Payment Settings ↗

✓ Optional: Add PayPal

You can also add PayPal as a second payment option on the same page.

💡 You can skip this step and come back later. The cart will work in Test Mode without a payment gateway connected.
Step 4 of 5

Connect Your Domain

Add these 2 DNS records at your domain registrar to point your domain to the website.

A @ 76.76.21.21
CNAME www cname.vercel-dns.com

Open your domain registrar:

Cloudflare Namecheap GoDaddy Google Domains Hostinger

Verify DNS (optional):

⏳ DNS changes take 5 minutes to 48 hours to propagate. You can check back later.
Step 5 of 5

Go Live Checklist

Review everything before launching your store.

⏳ Snipcart API Key

Checking...

💳 Payment Gateway

Verify at Snipcart → Payment Gateway that Stripe or PayPal is connected.

🌐 Domain & DNS

After DNS propagates, add your domain in Snipcart → Domains & URLs too.

🚀 Switch to Live Mode

When ready for real payments: in Snipcart toggle from Test to Live. Copy the new Live Public API Key and go back to Step 2 to update it.

🎉 That's it! Your GIFTELIX store is ready. Customers can browse products, add to cart, and checkout securely.
